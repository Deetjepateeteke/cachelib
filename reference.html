cachelib.MemoryCache¶

class cachelib.MemoryCache(name: str = '', max_size: int | None = None, max_memory: str | int | None = <cachelib.utils.NullValue object>, eviction_policy: ~cachelib.eviction.EvictionPolicy | None = None, ttl: int | float | None = None, verbose: bool = False, thread_safe: bool = True)[source]¶

Bases: BaseCache

An in-memory cache with lru/lfu eviction and optional ttl-support.

A context manager to enable read-only mode. The cache cannot be modified while read-only mode is enabled.

Usage:

set_read_only(read_only: bool) → None¶

Enable or disable read-only mode manually. The cache cannot be modified while read-only mode is enabled. It is preferred to use cache.read_only() as a context manager.

Parameters:

read_only (bool) – True to enable read-only mode, False to disable read-only mode.

Returns:

None

Usage:
>>> cache.set_read_only(True)  # Enable read-only mode
>>> ... some code while cache is read-only
>>> cache.set_read_only(False)  # Disable read-only mode
Raises:

CacheConfigurationError – When read_only isn’t a bool.
